# GraphCaster

Легковесный редактор и рантайм для сценариев в виде направленных графов: UI (веб / встраивание в приложение) и движок выполнения на Python.

## Структура

| Путь | Назначение |
|------|------------|
| `python/` | Пакет `graph_caster`: загрузка документа графа (JSON), обход, условия на рёбрах, события для UI/консоли |
| `ui/` | Фронтенд-заготовка (полотно, ноды, пины, панели) — см. `ui/README.md` |
| `schemas/` | Примеры и черновики формата документа графа |
| `doc/` | Продуктовый дизайн и план разработки |

## Быстрый старт (Python, локально)

```bash
cd python
pip install -e .
python -m graph_caster --help
```

Подробности архитектуры и UX: `doc/PRODUCT_DESIGNE.md`. Порядок фич: `doc/DEVELOPMENT_PLAN.md`.
